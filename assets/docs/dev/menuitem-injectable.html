MenuItemService

Description

This @Injectable() makes it easier to add menu items to a toolbar, similar to like Android. (For those of you who don't know, those icons on that toolbar are known as menu item just shown as action buttons and the toolbar's also known as an ActionBar.) Anyways, enough of Android and Material Design and whatever. In the future, this will be revisited and may be integrated with SharedInjectable.

Notices

Ensure that MenuItemService is imported to your main app's module under the providers array:

import { MenuItemService } from 'path/to/menuitem.service';
@NgModule({
    providers: [MenuItemService]
})

That's it! There's nothing else there is! On to the documentation then...

(Examples will be at the bottom this time...)

/**
 * Gets the menu items (getter)
 * @returns {MenuItem[]} The menu items
 */
get menuItems: MenuItem[] {
    return this.menuItemService.getMenuItems();
}

Adding menu items

export class MyComponent implements OnInit {
    constructor(private menuItemService: MenuItemService){}
    ngOnInit() {
        this.menuItemService.addMenuItem({title: "Reload", icon: "refresh", onClickListener: (ev: Event)=> {
            console.log("Reload was clicked");
            window.location.reload(true);
        }})
    }
}

Adding onclick listeners to a menu item

export class MyComponent implements OnInit {
    constructor(private menuItemService: MenuItemService){}
    ngOnInit() {
        this.menuItemService.addMenuItem({title: "Open sidenav", icon: "menu"});
        this.menuItemService.addMenuItemOnClickListener(0, (ev: Event)=> {
            console.log("Clicked");
        })
    }
}
