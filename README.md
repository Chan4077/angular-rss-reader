# Serving locally
## Developer mode
Run `ng serve` and go to `http://localhost:4000`.
## Service worker
To enable the service worker, run the following commands:
```bash
npm run build
cd dist
http-server
```
Go to `http://127.0.0.1:8080` to see it in action.
